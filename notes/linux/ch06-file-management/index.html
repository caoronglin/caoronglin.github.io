PATH 环境变量

什么是 PATH

PATH 是一个环境变量，指定了系统查找可执行程序的目录列表。当你在命令行输入一个命令时，系统会在 PATH 指定的目录中查找对应的程序。

ls 命令详解

ls (list) 命令用于列出目录内容，是使用最频繁的命令之一。

常用选项

选项 全称 说明
-l long 使用长列表格式（详细信息）
-a all 显示所有文件，包括隐藏文件（以.开头）

-R recursive 递归列出子目录内容
-F classify 在文件名后添加类型指示符（*/=@|）

文件内容查阅

直接检视文件内容

cat (concatenate)

将文件内容输出到标准输出（屏幕）。

⚠️ 注意：cat 适合查看小文件，大文件应该使用 less 或 more。

tac

反向输出文件内容（从最后一行开始）。

nl (number lines)

添加行号显示文件内容，比 cat -n 更灵活。

more

分页显示文件内容，适合查看大文件。

在 more 中的操作：

  • 空格 - 下一页
  • Enter - 下一行
  • b - 上一页
  • /pattern - 搜索
  • q - 退出
  • = - 显示当前行号

⚠️ 注意：more 不能向前翻页（只能用 b），现在一般使用 less。

less (推荐)

更强大的分页显示工具，比 more 功能更丰富。

在 less 中的操作：

按键 功能
↓ / Enter / e / j 向下滚动一行
↑ / y / k 向上滚动一行
空格 / Page Down / f 向下翻页
b / Page Up 向上翻页
d 向下翻半页
u 向上翻半页
→ / ← 水平滚动（当使用 -S 时）
g / < / Home 跳转到文件开头
G / > / End 跳转到文件结尾
= / Ctrl+G 显示当前位置信息
/pattern 向下搜索
?pattern 向上搜索
n 跳到下一个匹配
N 跳到上一个匹配
&pattern 只显示匹配的行
: 进入命令模式
-N 切换行号显示
q 退出
Q 退出，不保存标记
h 显示帮助
